## Matrix Policy

This matrix records source rights and availability for official/public GodView candidates. It is not an implementation plan, provider approval, data-ingestion approval, or source of alpha evidence.

All future work must keep these phases separate:

```text
source audit -> source policy -> tiny fixture approval -> provider proof -> later integration decision
```
