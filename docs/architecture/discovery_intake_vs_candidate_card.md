# Discovery Intake vs Candidate Card

Status: G8.1A object-boundary contract
Authority: Phase 65 G8.1A Discovery Drift Correction
Date: 2026-05-10

## Object Boundary

```text
Discovery intake item = this name may belong in the discovery universe.
Candidate card = this name has one structured thesis object.
Validated thesis = this name passed later evidence gates.
Actionable state = this name passed later thesis, behavior, and risk gates.
```

G8.1A only governs discovery intake items and their origin labels.

## Current Object Truth

`MU` is the only existing candidate card.

`DELL`, `INTC`, `AMD`, `LRCX`, and `ALB` remain intake-only.

No current intake item is validated or actionable.

## Promotion Rule

An intake item can only become a candidate card in a separately approved phase such as G8.2. A discovery-origin label is not promotion authority.

## Research Capture Rule

Research capture and source leads are not canonical evidence. They may point to later official-source work, but they cannot satisfy validation by themselves.
